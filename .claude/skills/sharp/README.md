## Key Features

- High-performance resizing (4-5x faster than ImageMagick)
- Format conversion (JPEG, PNG, WebP, AVIF, GIF, TIFF, HEIF)
- Smart cropping (entropy, attention/face detection)
- Color adjustments (brightness, saturation, grayscale)
- Effects (blur, sharpen, normalize)
- Compositing (watermarks, overlays, text)
- Animated GIF/WebP support
- Metadata preservation (EXIF, XMP, IPTC)
- Stream and buffer support
- Clone pipelines for parallel processing
- Batch processing utilities
